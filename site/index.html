An open-source AI coding agent that plans, writes, tests, and fixes code from natural language — with persistent memory, multi-model support, and CI/CD automation.

View on GitHub Contribute

Quick Install

npm install -g @signhify/cli
Then run signhify wizard for first-time setup.
🤖

Multi-Agent Modes

Build (full access), Plan (read-only), Debug, and Compose modes with per-mode tool permissions.

🧠

Persistent Memory

SQLite + FTS5 with automatic checkpoints and dream/distill self-improvement. Remembers your project across sessions.

🔌

Multi-Provider

OpenAI, Anthropic Claude, Google Gemini, or any OpenAI-compatible endpoint. Bring your own key.

⚡

Two-Tier Model Routing

Fast model for inline autocomplete, reasoning model for the agent loop — configured independently.

🎯

Goal/Judge Verification

Set a stop condition like "all tests pass" — verified by an independent model before the session terminates.

🔄

CI/CD Mode

Run with --auto --output json for scripted pipelines. Structured JSON output with exit codes.

🔍

VS Code Extension

Full chat panel + inline autocomplete. Shares the same core engine as the CLI.

🧩

MCP Extensible

Model Context Protocol client for custom tools, resources, and prompts from third-party servers.
