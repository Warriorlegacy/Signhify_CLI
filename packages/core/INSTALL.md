# @signhify/core — Installation

Agent runtime — loop, modes, permissions, context management.

## Install

```bash
npm install @signhify/core
```

## Requirements

- Node.js 20+
- No native dependencies

## Usage

```ts
import { AgentLoop, ModeManager, PermissionEngine } from '@signhify/core';
```

## Package scripts

```bash
npm run build
npm run dev
npm run typecheck
npm run clean
```
